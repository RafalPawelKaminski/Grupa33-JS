## _Operatory, warunki, funkcje_

Wprowadzenie do podstaw JS podstawowych działań na kodzie oraz pierwsze funkcje

## Operatory:

Artmetyczne:
- Dodawanie - +
- Mnożenie - * 
- Dzielenie - /
- Modulo - % 
- Potęgowanie - **
- Inkrementacja - ++
- Dekrementacja - --

Porównanie:

- '== oraz ==='
- '!= oraz !=='
- '> oraz <'
- '>= oraz <='

Logiczne:

- && - koniunkcja
- || - alternatywa
- ! - negacja
- ^ - alternatywa rozłączna
